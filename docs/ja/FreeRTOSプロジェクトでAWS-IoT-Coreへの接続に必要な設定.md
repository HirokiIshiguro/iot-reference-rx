# はじめに
本項では、RXファミリ向けFreeRTOSプロジェクトでAWS IoT Coreに接続するための方法を紹介します。

# RXファミリ向けFreeRTOSプロジェクトの設定方法
AWS IoT Coreへの接続には、証明書及び鍵情報が必要となります  
FreeRTOSプロジェクトに証明書及び鍵情報を設定する方法は、以下いずれかでご確認ください
* サンプルコード・アプリケーションノート(APN)をご利用の場合、同梱のドキュメントを参照ください
* RX向けFree RTOSプロジェクトの最新の利用方法は[Getting Start Guide](https://github.com/renesas/iot-reference-rx/blob/main/Getting_Started_Guide.md)を参照ください
